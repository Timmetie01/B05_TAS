# B05_TAS
Python code used by group B05 in Test, Analysis &amp; Simulation. 

Main feature:
Data class: contains all data regarding for a certain take.
Using all kinds of functions (central difference derivative, moving averages filter, least squares fitting, threshold filters etc) the data can be manipulated in many ways.
Using function in the Data class the results can then be plotted in many different ways (3D, 2D, 1D vs time, Scatter/line, colors, labels etc).
